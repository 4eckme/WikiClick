Свободный IT-обозреватель

WikiClick.Ru — это свободная веб-энциклопедия по сайтам, интернет-сервисам, приложениям, играм и любым другим разработкам, созданых программистами России, СНГ и других стран. Без цензуры и монополии на контент. Создание и редактирование страниц доступно каждому посетителю. Единственное условие публикации - соответствие тематике сайта.
WikiClick приглашает авторов и соавторов IT-проектов.
Статья на WikiClick даст другим людям узнать о Вашей работе, кроме этого вы получите бэклинки, что со временем продвинет Вас выше в поисковых системах.
Это свободная площадка, и если Вы не могли попасть на крупные wiki-проекты из-за монополии на контент, то теперь Вы можете рассказать о своей разработке здесь.
Open Source Wiki Engine
WikiClick - это еще компактный wiki-движок основанный на html, с открытым исходным кодом. Он хорошо работает с русским языком и метатегами, а также имеет валидную верстку, очень быстро загружает страницы, не содержит дубликаты страниц с похожими url и включает в себя настроенный robots.txt, - благодаря этим качествам он хорошо индексируется поисковыми системами в русском сегменте. GitHub проекта.
В CMS сохраняются предыдущие версии страниц, есть предварительный просмотр, можно делать постраничную выборку статей по разделам, тегам и по поисковому запросу, есть умный журнал правок (или журнал версий, который объединяет множественные правки от одного пользователя в одну весию) и журнал древовидных комментариев, подключена каптча для защиты от автоматического постинга, а также движок сам генерирует оглавление (или навигацию) для страниц, добавляя блок с деревом ссылок на заголовки (которые в частности могут содержать скрытый, выпадающий по клику, контент). Установлена подсветка для публикуемого кода, разрешена стилизация DOM-элементов по средствам атррибута style, и есть удобная загрузка картинок (в том числе превьюшек для разделов, тегов и поиска по сайту). Есть поддержка пользовательских закладок. Админка для управления контентом прилагается.
